# TiroVida v0.1 — Infrastructure & Auth

¡Primera versión de TiroVida! 🎉

## ¿Qué incluye?

### 🗄️ Backend
- Base de datos PostgreSQL con 8 tablas
- Row Level Security (RLS) completo
- Triggers automáticos (auto-crear perfil, timestamps)
- 12 categorías de comunidad pre-cargadas
- Migraciones con Supabase CLI

### 🔐 Autenticación
- Login con email/password
- Registro con seudónimo
- Google OAuth
- Confirmación por email
- Sesiones con cookie-based approach

### 🎨 Frontend
- Design System "Ethereal Sanctuary" (glassmorphism, lavanda-mint)
- Landing page con hero animado
- Dashboard con stats (racha, energía, último TSH)
- Navegación top + bottom (mobile)
- Páginas placeholder para todos los módulos

### 📚 Documentación
- Documentación completa con Docusaurus 3
- 4 secciones: Backend, Frontend, Guía de Usuario, Administración
- 30+ páginas de documentación
