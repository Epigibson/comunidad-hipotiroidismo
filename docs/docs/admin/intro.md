# 🔒 Panel de Administración

Guía para administradores y moderadores de TiroVida.

## Roles del sistema

| Rol | Descripción | Permisos |
|-----|-------------|----------|
| `patient` | Usuario estándar | CRUD propio, comunidad |
| `family` | Familiar/cuidador | Igual que patient |
| `moderator` | Moderador de comunidad | Gestión de contenido |
| `specialist` | Profesional de salud | Badge verificado |

## Herramientas de administración

- 👥 [Gestión de Usuarios](./user-management) — Roles, verificación, suspensiones
- 🛡️ [Moderación de Contenido](./content-moderation) — Reportes, bloqueos
- 📂 [Categorías de Comunidad](./community-categories) — Gestión de foros
- 📊 [Analytics](./analytics) — Métricas de uso
- 🚀 [Deployment](./deployment) — Despliegue y mantenimiento
