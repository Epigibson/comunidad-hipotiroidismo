# 📊 Reportes Clínicos

Los reportes de TiroVida correlacionan tus síntomas diarios con tus resultados de laboratorio para crear una vista integral que puedes compartir con tu endocrinólogo.

## ¿Qué incluye un reporte?

1. **Resumen de síntomas** — Promedios de energía, niebla mental, ánimo y dolor articular en un período
2. **Gráfica de tendencias** — Cómo han variado tus síntomas en el tiempo
3. **Resultados de laboratorio** — TSH, T3, T4 con rangos de referencia
4. **Correlación** — Líneas temporales superpuestas de síntomas vs. labs
5. **Historial de medicación** — Dosis y adherencia

## ¿Cómo generar un reporte?

1. Ve a **Reportes** desde el dashboard
2. Selecciona el período (últimos 30, 60, 90 días o personalizado)
3. El reporte se genera automáticamente
4. Puedes descargarlo como **PDF** para compartir con tu médico

## ¿Para qué sirve?

:::tip Para tu cita médica
Lleva el PDF de tu reporte a tu próxima cita con el endocrinólogo. Esto le ayuda a ver:
- Si tu medicación actual está funcionando
- Si hay patrones que requieren ajuste de dosis
- Cómo correlacionan tus labs con cómo te sientes
:::

## Privacidad del reporte

- El reporte se genera **localmente** en tu navegador
- **No se almacena en nuestros servidores**
- Solo tú decides con quién compartirlo
