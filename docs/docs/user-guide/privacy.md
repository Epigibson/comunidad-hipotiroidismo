# 🔒 Privacidad y Seguridad

TiroVida toma tu privacidad muy en serio. Tus datos de salud son **tuyos** y solo tuyos.

## ¿Qué datos almacenamos?

| Dato | Quién puede verlo |
|------|-------------------|
| Email | Solo tú y el sistema |
| Alias/seudónimo | Tú y la comunidad |
| Registros de síntomas | Solo tú |
| Resultados de laboratorio | Solo tú |
| Condiciones de salud | Solo tú |
| Posts en la comunidad | La comunidad (anónimo si prefieres) |
| Perfil público | Alias y avatar visibles para la comunidad |

## ¿Cómo protegemos tus datos?

### Row Level Security (RLS)
Cada consulta a la base de datos pasa por **políticas de seguridad** que verifican tu identidad. Es literalmente imposible que otro usuario acceda a tus datos de salud.

### Encriptación
- La conexión usa **HTTPS** (TLS 1.3)
- Las contraseñas se almacenan **hasheadas** (nunca en texto plano)
- Los tokens de sesión se renuevan automáticamente

### Sesudónimos
- Puedes usar un nombre ficticio en la comunidad
- Los posts anónimos no muestran tu alias

## Tus derechos

- ✅ Puedes ver todos tus datos en cualquier momento
- ✅ Puedes editar o eliminar tus datos
- ✅ Puedes eliminar tu cuenta por completo
- ✅ Puedes exportar tus datos como PDF

## Sin publicidad, sin venta de datos

:::info Sin fines comerciales
TiroVida es un proyecto comunitario. **No vendemos datos**, no mostramos publicidad, y no compartimos información con terceros.
:::
