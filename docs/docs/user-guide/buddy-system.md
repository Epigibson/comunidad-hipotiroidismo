# 🤝 Sistema Buddy

El sistema Buddy te conecta con otra persona que comparte tu diagnóstico para apoyarse mutuamente.

## ¿Cómo funciona?

1. **Solicitud**: Envías una solicitud de buddy a alguien de la comunidad
2. **Aceptación**: La otra persona acepta o declina
3. **Conexión**: Si acepta, se forma el match y pueden comunicarse
4. **Finalización**: Cualquiera puede terminar el match en cualquier momento

## Estados del match

| Estado | Significado |
|--------|-------------|
| `pending` | Solicitud enviada, esperando respuesta |
| `active` | Match activo, ambos conectados |
| `declined` | Solicitud rechazada |
| `ended` | Match terminado por alguno |

## Recomendaciones

:::tip Sé paciente
No todas las solicitudes serán aceptadas, y eso está bien. Respeta las decisiones de los demás.
:::

:::tip Establece límites
Tu bienestar es primero. Si una relación de buddy no te hace bien, puedes terminarla sin dar explicaciones.
:::
