# 🎨 Design Tokens

TiroVida usa el design system **Ethereal Sanctuary** — una paleta etérea con tonos lavanda, mint y cristal.

## Paleta de colores

### Colores primarios

| Token | Valor | Uso |
|-------|-------|-----|
| `--color-primary` | `#6a4aa2` | Botones, links, acentos |
| `--color-primary-dim` | `#5a3d8a` | Hover states |
| `--color-primary-container` | `#ede5f5` | Badges, backgrounds sutiles |
| `--color-on-primary` | `#ffffff` | Texto sobre primary |

### Colores secundarios (Mint clínico)

| Token | Valor | Uso |
|-------|-------|-----|
| `--color-secondary` | `#29655c` | Elementos médicos, labs |
| `--color-secondary-container` | `#d5f0eb` | Background de labs |

### Colores terciarios (Lavanda cálido)

| Token | Valor | Uso |
|-------|-------|-----|
| `--color-tertiary` | `#8b6f9d` | Badges, categorías |
| `--color-tertiary-container` | `#f0e8f5` | — |

### Semánticos

| Token | Valor | Uso |
|-------|-------|-----|
| `--color-success` | `#22c55e` | TSH normal, medicación OK |
| `--color-warning` | `#f59e0b` | TSH elevado, avisos |
